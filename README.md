# 프로젝트 개요
* 문제 출제자의 편의를 돕기 위한 문제 생성기입니다.
* 문제 출제자 뿐만 아니라 학습자도 본인의 학업 성취도를 평가하기 위하여 사용할 수 있습니다.
* [직접 해보기](quizgen.streamlit.app)

# Runs
```python
streamlit run app.py
```
